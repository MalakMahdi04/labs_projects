# Python Fundamentals Project

In this project, you will use fundamental Python skills to create an interactive application. Each team will be assigned an application to create based on user requirements.

## Deliverables

Your public repository must include the following:

- **Python file** (`.py`): A module containing the class that implements all functionalities outlined in the User Stories and Technical Requirements.
- **Jupyter Notebook** (`.ipynb`): Imports the class from the `.py` file and demonstrates its functionalities.
- **Presentation slide deck**: Introduces the application and highlights its key features.

## Submission

Submit the link to your repository on Canvas (one submission per group).
